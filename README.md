# 🚀 JobTracker

A personal job application and referral tracking dashboard built with React, Vite, and Google Sheets as the backend.

## Live App

🔗 https://abmax777.github.io/job-tracker/

## Features

- **Applications** — Track job applications across LinkedIn, Naukri, Instahyre and more
- **Referrals** — Log referral outreaches with daily 3/day goal tracker and follow-up reminders
- **Interviews** — Track interview rounds, outcomes and upcoming schedules
- **Dashboard** — Pipeline overview, charts, recent activity and follow-up alerts
- **Google Sheets backend** — All data syncs to a Google Sheet, accessible anywhere

## Tech Stack

- React + Vite
- Tailwind CSS
- Google Sheets API + Apps Script
- Recharts
- GitHub Pages (hosting)
